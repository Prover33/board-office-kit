## 二、未知主体解析（强制约定）
1. `WebSearch "<公司名> 股票代码"` 取权威结果（巨潮资讯 / 交易所 / 东方财富）。
2. 若同一昵称对应多家公司（如"华铁"可能指 603300，也可能被误指其他），列出候选让用户确认。
3. 拿到代码后再传给 `search_cninfo.py --code <代码>`；若只想"全市场搜某类行为"而不限定公司，则不传 `--code`，靠关键词 + `--type` 过滤。
4. 人名 / 产品名标准化同理（如"茅台"→600519）。

## 三、跨市场说明
- cninfo 覆盖沪(60/68 开头)、深(00/30 开头)、北交所(8/4 开头)全部 A 股。
- 港股(HKEX，5 位代码)、美股：cninfo 不包含，需另寻数据源，技能在报告中注明局限。
