# 查询拆解参考（借鉴 Wind AI 公告搜索底层原理）

来源：Wind 金融终端《自然语言需求拆解与关键词转换机制》技术文档（用户 2026-07-15 提供）。
本文将其"自然语言 → 结构化检索条件"的五阶段流水线，翻译为**本技能（cninfo 全文检索 + WebSearch 兜底）可落地**的版本。

> ⚠️ 能力边界：Wind 的「金融知识图谱 / 实体消歧 / ML 训练平台 / 标签体系」是千亿级基础设施，本技能**无法复刻**。
> 但「同义词扩展 + 多路并行检索 + 融合排序」这一**思路**已写入 `search_cninfo.py --expand`，可直接使用。

---

## 一、Wind 五阶段流水线（原文要点）

| 阶段 | Wind 做法 | 本技能对应 |
|---|---|---|
| 3.1 意图识别 | 按公司 / 按类型 / 按主题 / 按事件 / 混合 | 由用户措辞判断，决定把代码/类型词塞进检索式 |
| 3.2 金融实体识别(NER) | 昵称→标准代码，知识图谱消歧 | `references/entity_code_map.md` 映射（无图谱，未知主体走 WebSearch） |
| 3.3 维度提取 | 代码/类型/日期/标签等结构化条件 | `--code / --start / --end / --type` 后置过滤 |
| 3.4 关键词转换 | **5 条规则（见下）** | 实体标准化✅ 时间标准化✅ 类型同义词⚠️ 概念扩展✅(新增) |
| 3.5 多源融合 | 结构化+全文+标签+图谱，融合排序 | 同义词多路检索→按 announcementId 去重融合（已实现） |

## 二、关键词转换五规则（本技能落地情况）

- **规则一 实体标准化**：`茅台→600519`、`海南华铁→603300`。✅ 已落地（`entity_code_map.md`）。
- **规则二 公告类型同义词**：`买公司→收购/受让`、`罚钱→行政处罚/警示函`。⚠️ 部分落地（见 `search_cookbook.md`），未成体系表，后续可补。
- **规则三 时间表达标准化**：`最近→近1-3月`、`无限定→近6月`。✅ 已落地（`--start/--end`）。
- **规则四 概念/主题→标签**：`人工智能→[人工智能,AI,机器学习,大模型]`。✅ **新增**（`SYNONYMS` 含 `人工智能/新能源`）。
- **规则五 同义词扩展增强**：`收购→[收购,并购,购买,受让]`。✅ **新增**（`--expand`，见下）。

## 四、典型案例演示（照搬 Wind 三例，改为本技能检索式）

**案例一：单一公司 + 公告类型**
- 用户：「贵州茅台最近的分红公告」
- 拆解：实体=600519 / 意图=按公司+按类型(分红) / 时间=最近
- 本技能检索式：`search_cninfo.py "600519 分红" --start 2026-04-15`（"分红"经 --expand 覆盖 派息/利润分配）

**案例二：主题/概念搜索**
- 用户：「人工智能相关的上市公司公告」
- 拆解：概念=人工智能 → 标签+关联词扩展
- 本技能检索式：`search_cninfo.py "人工智能"`（--expand 自动补 AI/机器学习/大模型）

**案例三：复杂混合条件**
- 用户：「海南华铁近一年关联交易公告」
- 拆解：实体=603300 / 类型=关联交易 / 时间=近一年
- 本技能检索式：`search_cninfo.py "603300 关联交易" --start 2025-07-15 --end 2026-07-15`

## 五、对董办使用本技能的启示（取自 Wind 第六节，已改写）

1. **用标准实体名**：尽量给公司全称或 6 位代码，别用模糊简称（提高实体命中）。
2. **用标准类型词**：说「年度报告」「重大资产重组」「关联交易」，别太口语化。
3. **明确时间窗**：给「2025 年」「近三个月」，别用「之前」「以前」。
4. **善用 --expand**：自然语言搜不到时，开同义词扩展召回更广；仍不理想再换核心词组合。
5. **正文搜不到的词换搜索引擎**：cninfo API 只索引标题+摘要（见 `search_cookbook.md` 限制说明），正文级短语（如「前次/再次收到告知书」）须走 WebSearch `site:cninfo.com.cn`。
6. **案例宁多勿少（≥20）**：每次检索目标至少 20 个相关案例才有统计与对照价值；凑不够时依次放宽 --type / 时间窗 / 开 --expand / 换更宽泛检索式。若全市场真实不足 20（如二次告知类极端稀缺），如实说明并附全部已找到结果，不硬凑。
