# Changelog

## 1.0.0 (2026-07-21)

- **首次发布**：通用上市公司董办专家 agent，开箱即用 + 渐进式搭建。
- **内置 announcement-drafter + announcement-case-analyzer 两项技能**：安装即能起草临时公告 docx、检索A股上市公司公告案例并生成带原文链接的分析报告。
- 内置**场景路由表**、**源能力矩阵**、**回答精度硬约束**、**批量核查纪律**、**经验泛化方法论**。
- 零配置即可干通用董办活；填【配置区】、建 IMA 知识库、接 MCP 后越懂你家公司。
- **工作区规划与文件归位**：首启引导按主题分区（00~09 + ZZ 缓存），成品/临时分离。
- 配套资产：示例配置、MCP 接入清单、五阶段搭建指南（docs/01~05）、档案管理方法论。
- 适配 WorkBuddy 插件格式（agent 型，双语 manifest）。
