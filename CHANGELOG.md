# Changelog

## 1.2.0 (2026-07-22)

- **首次搭建向导**：新增「首次搭建向导」五步流程（依赖自检 → 连接器探测 → 配置引导 → 工作区确认 → 能力就绪报告），把"提醒用户搭建"升级为 AI 首次调用时主动跑一遍环境检查，透明输出能力就绪报告（✅/⚠️/❌）。
- **配置区不填硬后果透明化**：明确必填项 `<TARGET_CO>` / `<TICKER>` 空缺时专家无法指代公司、公告起草/行情检索/F10 拉取全部不可用；各选填项"不填时的降级行为"已在表中列清。
- **公告样式 DNA 对齐母版实测**：修正样式铁律与场景路由描述——公告名居中无缩进、公告头左对齐、正文顶格（非缩进 2 字符）、一/二/三级标题首行缩进 2 字符（firstLineChars=200，非顶格）；与内置 announcement-drafter 的 `references/style_spec.md` 完全一致。

## 1.0.0 (2026-07-21)

- **首次发布**：通用上市公司董办专家 agent，开箱即用 + 渐进式搭建。
- **内置 announcement-drafter + announcement-case-analyzer 两项技能**：安装即能起草临时公告 docx、检索A股上市公司公告案例并生成带原文链接的分析报告。
- 内置**场景路由表**、**源能力矩阵**、**回答精度硬约束**、**批量核查纪律**、**经验泛化方法论**。
- 零配置即可干通用董办活；填【配置区】、建 IMA 知识库、接 MCP 后越懂你家公司。
- **工作区规划与文件归位**：首启引导按主题分区（00~09 + ZZ 缓存），成品/临时分离。
- 配套资产：示例配置、MCP 接入清单、五阶段搭建指南（docs/01~05）、档案管理方法论。
- 适配 WorkBuddy 插件格式（agent 型，双语 manifest）。
