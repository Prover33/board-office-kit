## 安装

**复制下面这段话发给你的 AI agent，它就会帮你装好：**

> 帮我把 board-office-kit 董办专家装到 WorkBuddy。执行：
>
> ```bash
> git clone https://github.com/Prover33/board-office-kit.git ~/.workbuddy/plugins/marketplaces/my-experts/plugins/board-office-kit
> ```
>
> 装完后在 WorkBuddy 的「专家」左侧栏 → 我的专家 → 找到 board-office-kit → 点击即可开始对话。

或者你也可以手动 git clone 到上面那个路径，然后在 WorkBuddy 专家面板启用。

---

一个通用、**开箱即用**的上市公司董办专家 skill。内置场景路由表、源能力矩阵、回答精度硬约束、大批量核查纪律、经验泛化方法论，并对接公告案例 / 知识库 / 工商 / 行情 / 法规 / 联网检索等数据源。**已内置 announcement-drafter（公告起草）+ announcement-case-analyzer（案例搜索）两个技能——安装即能起草可交付的交易所临时公告 docx、检索A股上市公司公告案例并生成带原文链接的分析报告，无需额外安装。**

> **越搭越精准**：零配置即可干通用董办活；填公司信息、建 IMA 知识库、接 MCP 连接器后，专家越来越懂你家公司——但**不需要等全配完才用**，先让它干活，边用边搭。

## 它能干什么

| 你问 | 它答 |
|---|---|
| 「关联交易要披露吗？上不上会？」 | 按上市规则给具体金额阈值+审议路径，标明法规条款 |
| 「帮我查现任董事会成员」 | 从通达信/公告拉名单，附任职日期与来源链接 |
| 「核查下属控股企业哪些设了董事会」 | 逐家核验，不漏网，结果汇入可溯源台账 |
| 「这笔对外投资要不要党委前置？」 | 若你填了国资链→自动走双体系；否则给通用规则 |
| 「帮我起草一份对交易所问询函的回复」 | 给骨架+要点清单，引用同类公告案例 |
| 「帮我起草一份董事会决议公告」 | 套母版/数据驱动生成可交付的临时公告 docx（内置 announcement-drafter，无需另装） |
| 「某行为会不会被认定为内幕交易？」 | 分析构成要件+法规依据+类似处罚案例 |
| 「帮我把董办档案目录理一理」 | 只读扫描→建议归类→生成目录索引；后续帮你快速找文档、存新文件 |
| 「先帮我规划一个工作区」 | 按主题预建分区（公文草稿/核查台账/案例与法规/知识库/档案目录/工作缓存）；之后生成文件自动归位，顶层不堆文件 |

| | 没有这个 skill | 装了（零配置） | 填好配置 + 接 MCP + 建 IMA 库 |
|---|---|---|---|

├── 05-会议记录/        三会会议材料、决议记录
├── 06-信披材料/        定期报告草稿、公告底稿、披露时间表

## 快速开始（3 步）

### 第 1 步：安装

将 `board-office-kit` 放入 WorkBuddy 的插件目录：

```bash
cp -r board-office-kit ~/.workbuddy/plugins/marketplaces/my-experts/plugins/
```

或在 WorkBuddy 内调用 **expert-manager** 技能 → 导入本目录。

### 第 2 步：最少填 2 项即可启动

打开 `agents/board-office.md`，在【配置区】「必填」里填：

- **公司全称** → 如 `示例智造股份有限公司`
- **证券代码** → 如 `SH/600000`

> 不知道怎么填？看 `examples/config.example.md`，有完整的虚构公司示范。

> 其余 6 项都是选填——每多填一项，某方面能力就升一级。**不填也能先用**。

### 第 3 步：开始用

```
用户：关联交易的披露门槛是多少？
专家：（给出具体金额阈值 + 条款号 + 来源链接）
用户：帮我核查下属控股企业董事会设置
专家：（按批量核查纪律逐家核验，汇入台账）
用户：帮我起草一份董事会决议公告
专家：（套母版/数据驱动生成可交付的临时公告 docx）
```

## 渐进式搭建（搭得越多越强）

搭完第 1 步就能用。后续每一步都让专家在某方面更准：

| 步骤 | 做什么 | 花多少时间 | 得到什么 |
|---|---|---|---|
| **0** | 啥都不填，直接问 | 0 分钟 | 通用董办助手（法规/公告/合规判断/公文骨架） |
| **1** | 填公司名 + 代码 | 1 分钟 | 专家知道"你是谁"，回答从通用变指向 |
| **2** | 补全板块/国资/治理架构 | 3 分钟 | 国资双体系生效、各板块差异规则准确 |
| **3** | 建 IMA 知识库，传制度 PDF | 15–30 分钟 | 制度/权限类问答命中内部规定，不再泛泛 |
| **4** | 接 MCP（企查查/通达信/北大法宝/Tavily） | 每个 5–10 分钟 | 工商/行情/法规从联网升级为权威源直查 |
| **5** | 接 announcement-case-analyzer 技能 | 5 分钟 | 公告案例检索带 19 类事件枚举+合并规则（announcement-drafter 已内置，无需此步） |

## 内置技能（无需另装）

- **announcement-drafter**（交易所临时公告起草）：已随包内置于 `skills/announcement-drafter/`，安装专家即拥有。覆盖「识别类型 → 扫历史同类 → 套母版/参考同业 → 数据驱动生成 docx → 过自检清单」全流程，固化信披样式 DNA（标题黑体红色、正文宋体小四 1.5 倍、一二级宋体加粗顶格）与 Word 字体回退防护。首次使用请按该技能 SKILL.md 顶部「首次配置」把母版/索引/归档路径改为你自己的环境。

## 依赖说明

本 skill **装好就能独立运行**，以下均为增强项（缺任何一个都不会报错，只是对应能力降级）：

| 缺什么 | 对应能力降级为 |
|---|---|
| 未装 `announcement-case-analyzer` | 公告案例改用 Tavily 联网搜索兜底 |
| 未接 IMA 知识库 | 制度/权限类问答走公开法规 + 通用口径 |
| 未接企查查/天眼查 | 工商股权问题给通用判断 + 手动获取路径 |
| 未接通达信 | 董监高/行情数据走公开网页 + 年报 PDF |
| 未接北大法宝 | 法规给通用条文 + 链接到官方全文 |

> **announcement-drafter 已内置**，无需单独安装；仅首次使用需按该技能 SKILL.md 把本机母版/索引/归档路径改为你自己的环境。
> 以上增强项均可通过 WorkBuddy 技能市场或 MCP 设置页逐个接入，无需一次配齐。
